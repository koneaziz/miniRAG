# 🤖 Mini RAG Chatbot – Projet Streamlit
## 🧠 Description
Ce projet est un **chatbot RAG (Retrieval-Augmented Generation)** développé avec **LangChain** et **Streamlit**.  
Il permet de **poser des questions sur le contenu de documents PDF** que l’utilisateur charge dans l’interface.

## 🚀 Fonctionnalités
- 📄 **Chargement de plusieurs fichiers PDF**
- ✂️ **Découpage automatique** du texte en segments exploitables
- 🔡 **Création d’un index vectoriel FAISS**
- 🔎 **Recherche contextuelle** dans les documents
- 💬 **Interface de chat** interactive avec historique
- 🧠 **Réécriture automatique** des questions en fonction du contexte (RAG moderne)
